智慧医养大数据公共服务平台医生服务系统
System Architecture Diagram  |  系统架构分层图  |  v1.0
客户端层 Client Layer 用户入口
医生用户
  • 浏览器访问（Chrome / Edge / Firefox）
  • 移动端 APP（iOS / Android）
  • 响应式布局自适应
Vue3 前端应用
  • ElementPlus 页面组件库
  • Axios 请求封装与拦截器
  • ECharts 工作台图表渲染
  • Vue Router 路由守卫
  • Pinia 状态管理
网关层 Gateway Layer 安全与路由
Nginx 反向代理
  • 请求转发与负载均衡
  • 静态资源托管与 Gzip 压缩
  • 跨域 CORS 配置
HTTPS 加密通信
  • SSL/TLS 证书配置
  • 数据传输加密保障
  • HTTP 强制跳转 HTTPS
SpringBoot 后端服务 Backend Services 核心业务
Controller 表现层
AccountController
个人账户相关接口：登录、注册、密码修改、短信发送
ElderArchiveController
健康档案接口：档案CRUD、体检记录、分页查询
Service 业务逻辑层
AccountService
密码校验、短信验证码处理、工作台数据统计汇总
ElderArchiveService
档案新增与编辑、分页查询组装、体检数据关联
Mapper 数据访问层 MyBatis-Plus
AccountMapper
医生账户表数据操作
ElderArchiveMapper
老人档案主表数据操作
ElderPhysicalMapper
体检记录子表数据操作
数据存储层 Data Layer 持久化存储
MySQL 8.0 关系型数据库
  • doctor_account — 医生账户表（账号、密码哈希、手机号、角色）
  • elder_archive — 老人档案主表（姓名、年龄、病史、关联医生）
  • elder_physical_record — 体检记录子表（血压、血糖、心率、时间）
  • health_warning — 健康预警表（异常指标、预警等级、处理状态）
Redis 6.2 高速缓存
  • 短信验证码缓存 — TTL 5分钟自动过期，防刷控制
  • 工作台统计数据缓存 — 档案总数、今日新增、预警计数
  • 用户会话信息 — Token 存储与黑名单管理
基础设施层 Infrastructure 配置与工具
application.yml 全局配置文件 数据源、端口、日志级别
CacheTask 定时任务 验证码定时清理 统计缓存预热刷新
安全与工具类 BCrypt 加密工具 RedisTemplate 操作封装
全局异常处理器 统一错误响应格式 PageHelper 分页工具
